# Base Grid

Base Grid는 링크, 도구, 메모, 작업, 프롬프트를 블록 단위로 배치해 목적별 워크스페이스를 만드는 개인용 모듈러 대시보드입니다.

## 열어보기

`index.html` 파일을 브라우저로 열면 바로 실행됩니다. 별도 설치나 서버 실행이 필요하지 않습니다.

## 현재 포함된 기능

- Home Base, Creator Desk, Utility Dock, Business Lab 워크스페이스 전환
- Link, Note, Task, Prompt, Utility 블록 샘플
- 새 블록 추가
- 링크 블록 URL 입력 및 새 탭 열기
- 블록 선택, 복제, 삭제
- Ctrl+K Alfred 스타일 검색 팔레트
- 블록 드래그 앤 드롭 정렬
- 중요 블록만 보는 Focus 모드
- 라이트/다크 테마 전환
- 브라우저 localStorage 저장
